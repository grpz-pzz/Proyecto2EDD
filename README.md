# Proyecto2EDD
Proyecto 2 de Estructura de Datos

El proyecto consiste en un administrador de colas de impresion donde se prioriza la impresion segun la prioridad del usuario y cantidad de documentos. Los sistemas que se utilizan son: Monticulo Binario, Tabla Hash y la librearia GraphStream.
